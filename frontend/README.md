# DeclanLScott.com

- **Framework**: [Astro](https://astro.build)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Deployment**: [Cloudflare Pages](https://pages.cloudflare.com)

## Running Locally

```bash
git clone https://github.com/declanlscott/DeclanLScott.com.git
cd DeclanLScott.com
pnpm install
pnpm dev
```
